# Excel → SQL Server → Power BI Data Project

## Project Overview
This project demonstrates an end-to-end data workflow using Excel, SQL Server, and Power BI.

## Tools Used
- Excel
- SQL Server Management Studio (SSMS)
- Power BI

## Process
1. Uploaded Excel file into SQL Server
2. Cleaned and transformed data using SQL queries
3. Connected SQL Server to Power BI
4. Built interactive dashboards

## SQL Work
- Joins and aggregations
- Business logic applied in SQL
